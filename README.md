An AI-powered Resume Analyzer and Career Intelligence Web Application built using Flask and React. The platform analyzes user skills, detects the most suitable tech role, scores job match percentages, and generates personalized AI career guidance with learning roadmaps.

Features
•	Skill Extraction from resume text
•	Automatic Role Detection
•	AI Job Match Scoring
•	Country-Based Company & Salary Intelligence
•	AI Career Coach Review
•	Personalized Learning Roadmap
•	Multi-Country Support
•	Direct Job Board Links (LinkedIn, Indeed, Naukri, Glassdoor, etc.)

Tech Stack

Backend
•	Python
•	Flask (>=3.0.0)
•	Regex-based Skill Extraction
•	AI-style Scoring Logic

Frontend
•	React (via CDN)
•	Modern CSS UI
•	Dynamic Job Cards
•	Interactive Learning Dashboard
